Welcome to the AngularBootstrapTemplate Website! This template is meant to get you started with creating an Angular application with Bootstrap.

Each page passes data to the layout components via the ILayoutDataService

There is also an IErrorAlertsService and component that aid with showing errors to users. Enter text and click "Generate Error" to see it in action. Errors are pretty customizable on creation. They can specify if/when to auto-dismiss, if they should fade, and what should happen when they are removed. The header component defines these errors to be displayed below the other header content with absolute positioning.
